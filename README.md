### 클래스, 객체, 인스턴스
* 클래스 - 인스턴스를 만들기 위한 틀, 속성과 행동을 갖는다.
* 인스턴스 - 클래스를 토대로 만들어진 객체, 속성을 갖는다. 행동은 클래스에 있다.
* 객체 - 실체, 관련된 것들의 모음.
